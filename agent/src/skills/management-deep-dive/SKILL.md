# 管理层纵深研究:买股票就是买人

对一个公司(或 `人名 公司名`)做管理层深度研究。

> "买股票就是买人。找到你信任的人,然后长期持有。" —— 段永平
>
> "评估管理层,要看他们在没人看着的时候做什么。" —— 巴菲特

## 设计理念

多数投研对管理层的评估停留在表面:履历、持股、薪酬。但巴菲特花大量时间和管理层交流,李录说投资的本质是投人,段永平说买股票就是买人。

AI 无法和管理层吃饭,但可以通过公开信息做到:① **追踪管理层的话与做是否一致**(承诺 vs 兑现);② **分析每一笔重大资本配置决策的回报**;③ **从困难时期的决策推断品格**;④ **通过员工/商家/客户反馈侧面验证**。

适用场景:标准研究中管理层评分不确定(★★★ 或以下)、或管理层是核心投资逻辑时,用本技能做纵深。

## 第一步:识别关键管理层

用 `web_search` / `get_stock_profile` / `get_sec_filings` 确认关键人物:

| 角色 | 姓名 | 任期 | 背景 | 持股/期权 |
|------|------|------|------|----------|
| CEO/董事长 | | | | |
| CFO | | | | |
| 创始人(如不在位) | | | | |
| 实际控制人(如不同于 CEO) | | | | |
| 其他关键高管 | | | | |

**区分"谁在做决策"和"谁的名字在头衔上"**——有些公司创始人虽卸任仍是灵魂人物(如黄峥之于拼多多)。

然后收集四类数据(可顺序用 `web_search` + `get_stock_news` + `get_financial_statements` + `get_sec_filings`,或用 `run_swarm` 并行):① CEO 公开发言与预测记录;② 资本配置决策(并购/回购/分红/新业务);③ 治理结构与薪酬;④ 侧面验证(员工/客户/行业口碑)。

## 第二步:CEO 能力圈评估

### 战略眼光

用 `web_search` 搜 CEO 过去 5 年公开发言(股东信、电话会、采访、社交媒体),提取其判断:

| 时间 | CEO 的判断/预测 | 实际结果 | 准确度 |
|------|----------------|---------|:------:|
| | "我们认为 X 市场会..." | X 市场实际... | ✅/❌ |

关键问题:有没有做过超前于市场的正确判断?有没有在大家都看好时保持冷静?对行业的理解是跟随市场还是独立思考?

### 执行能力

| 维度 | 评估 | 证据 |
|------|------|------|
| 战略到落地 | 说了的事做到了吗? | |
| 组织能力 | 能吸引和留住人才? | |
| 危机处理 | 遇到困难时怎么应对? | |
| 迭代速度 | 犯错后纠正的速度? | |

## 第三步:诚信度评估(最重要)

> 巴菲特:"我们寻找三种品质:正直、智慧、精力。如果没有第一种,后两种会害死你。"

### 承诺 vs 兑现追踪

从过去 3 年财报电话会、股东信、公开采访中(`get_sec_filings` / `web_search` / `get_stock_news`),提取管理层做过的**具体承诺**:

| # | 时间 | 承诺内容 | 承诺场合 | 兑现情况 | 评价 |
|---|------|---------|---------|---------|------|
| 1 | | "2025 年实现 X 业务盈利" | 2024 年报电话会 | | ✅/⚠️/❌ |

| 承诺兑现率 | 评价 |
|:---------:|------|
| >80% | 优秀——说到做到 |
| 60-80% | 合格——大方向对,执行有偏差 |
| 40-60% | 令人担忧——承诺过多,交付不足 |
| <40% | 严重问题——不可信赖 |

### 困难时期的表现

搜公司历史上的重大危机(股价暴跌、业绩 miss、监管冲击、竞争加剧),分析管理层应对:是主动沟通还是躲避?归因内部还是甩锅外部?趁机做困难但正确的事,还是短期讨好市场?

### 对利益相关方的态度

| 利益相关方 | 管理层态度 | 证据 | 评价 |
|-----------|-----------|------|------|
| 股东 | 尊重/忽视/利用 | | |
| 员工 | 善待/压榨/漠视 | | |
| 客户/用户 | 以客户为中心/短期榨取 | | |
| 商家/供应商 | 公平合作/极端压价 | | |
| 监管/社会 | 合规配合/打擦边球 | | |

> 李录:"对利益相关方的态度决定了企业的长期生命力。短期压榨能提升效率,但长期会损害生态。"

## 第四步:资本配置能力

这是巴菲特最看重的——**每赚一块钱,管理层能把它变成多少钱?**

搜索过去 5 年重大资本配置决策(`web_search` / `get_financial_statements`),逐笔评估:

**并购记录**

| 时间 | 收购标的 | 金额 | 战略逻辑 | 事后回报 | 评分(1-5) |
|------|---------|------|---------|---------|:---------:|

**回购记录**(用 `financial_rigor` 的 `command=verify_valuation` 校验回购时与当前的 PE 等)

| 时间 | 回购金额 | 平均回购价 | 当时 PE | 事后回看 | 评分(1-5) |
|------|---------|-----------|:------:|---------|:---------:|

**分红记录**

| 年份 | 分红金额 | 分红率 | 同期 FCF | 是否可持续 |
|------|---------|:------:|---------|:---------:|

**新业务投资**

| 时间 | 投资领域 | 累计投入 | 当前状态 | 回报评估 | 评分(1-5) |
|------|---------|---------|---------|---------|:---------:|

### 资本配置评分

| 维度 | 评分(1-5) | 说明 |
|------|:---------:|------|
| 并购纪律 | | 是否在合理价格收购?整合如何? |
| 回购时机 | | 低估时回购、高估时停止? |
| 分红合理性 | | 分红率是否与 FCF 匹配? |
| 新业务投资 | | 成功率?止损纪律? |
| 现金管理 | | 现金储备合理?是否囤积过多? |
| **综合** | | |

> 巴菲特标准:理想管理层有好机会时果断投资,没好机会时积极回购/分红,永远不做高价并购。

## 第六步:侧面验证

AI 无法面对面交流,但通过公开渠道侧面验证。**注意**:取决于公开可搜索的内容,可能不完整,标注来源和可得性。

- **员工视角**:`web_search` 搜 Glassdoor 评分摘要、知乎讨论等(脉脉等需登录的标注"用户可自行补充")——企业文化、管理层评价、工作强度、薪酬满意度、发展前景。
- **客户/商家视角**:App Store 评分趋势、消费者投诉、商家论坛——产品满意度、客户服务、商家关系。
- **行业口碑**:同行和业内人士对该管理层的评价。

## 第七步:CEO 离开后的情景分析

> 巴菲特:"好公司应该是傻瓜都能经营的——因为迟早会有傻瓜来经营。"

- 如果 CEO 明天离开,公司能正常运转吗?
- 现有管理团队深度如何?有明确继任者吗?
- 竞争优势依赖 CEO 个人,还是组织/系统?
- 历史上的管理层交接是否顺利?

## 第八步:输出管理层评估报告

结构:① 关键人物速览;② 诚信度评估(承诺兑现率 + 困难时期 + 利益相关方);③ 能力评估(战略眼光 + 执行 + 资本配置);④ 治理结构;⑤ 侧面验证;⑥ 综合评分与结论。

### 综合评分

| 维度 | 权重 | 评分(1-5) | 加权 |
|------|:----:|:---------:|:----:|
| 诚信度 | 35% | | |
| 战略与执行 | 25% | | |
| 资本配置 | 25% | | |
| 治理结构 | 15% | | |
| **综合** | 100% | | |

### 段永平的"买人"三问

1. 这个人是否**正直**?(诚实、不占股东便宜)
2. 这个人是否有**能力**?(战略眼光 + 执行力 + 资本配置)
3. 你愿意把钱交给这个人管 **10 年**吗?

三个"是" = ★★★★★;前两个"是" = ★★★★;只有第一个"是" = ★★★;第一个不是"是" = ★(不投)。

## 第九步:保存报告

用 `write_file` 写入 `reports/{公司}-management-{YYYYMMDD}.md`。

完成后可用 `report_audit`(`extract` → 逐点核验 → `verdict`)对报告里的数字做准出门。

## 关键原则

- **诚信是一票否决项**——能力不足可以学,品格有问题无法修。
- **看行为不看言辞**——说什么不重要,做了什么才重要。
- **在困难中看真相**——顺风时谁都是好 CEO,逆风时才见真功夫。
- **资本配置是终极考试**——赚钱容易,把赚到的钱配置好难。
- **不要爱上管理层**——保持客观,即使你欣赏的人也可能犯大错。
